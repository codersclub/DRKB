Что такое фреймы?
=================

::: {.date}
01.01.2007
:::

Фрэйм - это что-то типа TForm, только его можно размещать в любом окне
программы.

Надо создать его: File -\> New -\> Frame.

Появится окошко. Растянуть на форме этот TFrame,
выбрать из списка нужный, и работать как с обычной формой.

Все просто.
