Как мне избавиться от выскакивающего окна CPU при ошибках?
==========================================================

::: {.date}
01.01.2007
:::

Зайти в реестр:

HKEY\_CURRENT\_USER\\Software\\Borland\\Delphi\\4.0\\Debugging

Поменять значение:

ViewCPUOnException=0

Взято с сайта <https://www.delphifaq.com>
