Как поменять установки на case-sensitive у сервера
==================================================

Используйте Rebuild Master Utility, она же Binn\\rebuildm.exe.

Если память не подводит, то схема такая:

    backup всех баз -> shutdown -> rebuild -> startup -> restore.
