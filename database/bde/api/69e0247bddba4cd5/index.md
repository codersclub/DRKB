Управление транзакциями
=======================

::: {.date}
01.01.2007
:::

Each function listed below begins, ends, or inquires about the status of
a transaction.

DbiBeginTran:

Begins a transaction.

DbiEndTran:

Ends a transaction.

DbiGetTranInfo:

Retrieves the transaction state.

Взято с Delphi Knowledge Base: <https://www.baltsoft.com/>
