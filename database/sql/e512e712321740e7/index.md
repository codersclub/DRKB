Как получить результирующим полем разницу между хранимой датой и текущей датой?
===============================================================================

::: {.date}
01.01.2007
:::

    SELECT CAST((поле_с_датой -"NOW") AS INTEGER) FROM MyBase

Получишь результат в днях.
