Создание файлов ресурсов
=======================

Вы можете использовать компилятор ресурсов командной строки Borland (BRCC.EXE и BRCC32.EXE), который поставляется с Delphi и C++ Builder, или редактор ресурсов WYSIWYG, такой как Borland\'s Resource Workshop 5.0, который поставляется с Borland RAD Pack.

Resource Workshop 5.0 может компилировать и декомпилировать как 16-, так и 32-битные ресурсы из файлов .rc, .res, .exe, .dll, .drv, .vbx, .cpl, .ico,. bmp, .rle, .dlg, fnt и cur.
