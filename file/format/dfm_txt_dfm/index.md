DFM -> TXT, TXT -> DFM
======================

::: {.date}
01.01.2007
:::

Use the Convert.exe file found in the Delphi\\C++ Builder

bin directory.

Example:

C:\\Delphi\\convert enum.dfm enum.txt

C:\\Delphi\\convert enum.txt enum.dfm
