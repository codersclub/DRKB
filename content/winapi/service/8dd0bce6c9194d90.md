Как проверить, не запущена ли Terminal Client Session?
======================================================

::: {.date}
01.01.2007
:::

    function IsRemoteSession: Boolean;
    const
      sm_RemoteSession = $1000; { from WinUser.h }
    begin
      Result := GetSystemMetrics(sm_RemoteSession) <> 0;
    end;
