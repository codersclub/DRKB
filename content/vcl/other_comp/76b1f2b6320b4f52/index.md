Что такое фреймы?
=================

::: {.date}
01.01.2007
:::

Фрэйм - это чтото-типа TForm только его можна размещать в любом окне
программы.

Надо создать его File -\> New -\> Frame. Появится окошко. Растянуть на
форме этот TFrameБ выбрать из списка нужный и работать как с обычной
формой. Все просто.
