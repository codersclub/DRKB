В режиме отладки приложения не разрешается открытие базы данных
===============================================================

::: {.date}
01.01.2007
:::

Необходимо отключить (деинсталлировать через Oracle Installer) Trace
Service на клиенте - совет от ORACLE.  Глюк имеет место быть только под
Windows NT 4.xx.
