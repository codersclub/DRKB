Как можно создать OCX-компонент?
================================

Для Delphi 6:

1. Пишешь обычный VCL-компонент и добавляешь его в палитру компонентов.
2. File -\> New -\> Other -\> ActiveX -\> ActiveX Control
3. VCL Class Name = твой VCL-компонент
