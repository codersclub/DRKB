Как добавить пункт в контекстное меню IE?
=========================================

В реестре по адресу:

    HKEY_CURRENT_USER
      SOFTWARE
        Microsoft
          Internet Explorer
            MenuExt

два параметра, посмотришь на другие - поймешь что они означают.
